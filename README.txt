This is a project that demo's how to use .Net Core 5 with EntityFramework to create APIs and perform CRUD operations on the database.
Angular 11 is used for the UI.

A project that allows you CRUD beverages and beverage information (stored in SQL Server)

How to setup this solution:

1) Install SQL server
2) Run the scripts in DBScripts folder
3) Install VS2019
4) Build WebAPIDemo project and run
5) Host the Angular frontend and browse to it

SQL Server DB name is Neptune, the Angular UI is called NeptuneUI
